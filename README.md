Projeto Cantina Fatec 2026.

Este é o meu projeto para as disciplinas de Linguagem de Programação II e Estrutura de Dados.(primeira avaliação).
Estou desenvolvendo um sistema "simples", (pois sou estudante do 2º semestre de IA na Fatec Rio Claro/SP), para gerenciar uma cantina escolar,
que atenda aos alunos do curso de IA, ESG, Professores e Colaboradores da Fatec.

O que o sistema faz?
O código utiliza Orientação a Objetos para organizar as informações da cantina, ou seja, controle de estoque, controle de pagamento,
controle do que consumiu, gerar e armazenar dados (utilizando a biblioteca Faker e Pickle), gerar relatorios de vendas e consumo.

As entregas serão incrementais via github, por enquanto, já criei:
A classe para cadastrar os produtos.
Arquivos do projeto:
cantinafatec2026.py: Arquivo principal com a lógica do sistema.
cantinafatec2026usuario.py: Arquivo que cuida dos dados dos usuários.
E, coloquei um código teste provisório.

Para rodar, basta ter o Python instalado e executar o arquivo principal.
Eu estou usando o VS Code para desenvolver e testar as funções no terminal.
